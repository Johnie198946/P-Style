# 📸 照片风格克隆应用 - 项目概览

## 🎯 项目背景

这是一个专业的照片风格分析和克隆工具，帮助摄影师和后期处理人员快速理解和复刻照片风格。通过 AI 分析源照片的风格特征，为目标照片生成详细的 Lightroom 和 Photoshop 调整方案。

## ✨ 核心功能

### 1. AI 智能分析

应用使用 AI 对照片进行多维度深度分析：

#### 🎨 构图分析（7个专业维度）
按照专业摄影师的标准进行全面的构图分析：

1. **画面主结构分析**
   - 几何框架识别（三分法、黄金分割等）
   - 视觉轴线分析（横向、纵向、对角线）
   - 平衡关系评估

2. **主体位置与视觉权重**
   - 主体位置判定
   - 面积占比计算
   - 黄金分割关系
   - 视觉权重评估

3. **线条与方向引导**
   - 引导线系统分析
   - 线条走向影响
   - 观众视线动线路径

4. **空间层次与分区**
   - 前景、中景、背景分层
   - 透视关系评估
   - 空间压缩与延展

5. **比例与留白**
   - 主体与留白比例
   - 留白分布分析
   - 元素密度评估

6. **视觉平衡与动势**
   - 水平与垂直元素关系
   - 稳定性与动态性
   - 视觉张力评估

7. **构图风格归类与改进建议**
   - 风格类型判定
   - 具体改进方向
   - 可操作的优化建议

#### ☀️ 光影参数
- **基础调整**：曝光、对比度、高光、阴影、白色、黑色
- **质感细节**：纹理、清晰度、去雾、饱和度、活力
- 所有参数都有具体的数值范围（如：+10 到 +18）

#### 🌈 色彩方案
- **白平衡**：色温（K值）、色调（Tint）
- **色彩分级**：
  - 高光色调（色相、饱和度）
  - 中间调色调
  - 阴影色调
  - 平衡值
- **HSL 调整**：每种颜色的色相、饱和度、明度
- **曲线调整**：Luma 曲线、RGB 各通道微调
- **色彩对比度**：具体的对比增强方法

#### 📷 Lightroom 方案
包含完整的 LR 面板参数：
- 基本面板（色温、色调、曝光等）
- 细节与质感
- 色彩调整
- HSL/颜色
- 色调分离
- 色调曲线

#### 🎨 Photoshop 方案
详细的 PS 后期步骤：
- Camera Raw 调整
- 色彩分级技巧
- 渐变映射应用
- 局部调整方法
- 最终优化流程

### 2. 两阶段分析流程

#### Stage 1: 基础分析
快速生成核心分析内容：
- ⭐ 照片点评
- 🎨 构图分析
- ☀️ 光影参数

用户可以先查看这些基础信息，决定是否需要深入分析。

#### Stage 2: 完整方案
点击"查看详细方案"后生成：
- 🌈 色彩方案
- 📷 Lightroom 完整参数
- 🎨 Photoshop 详细步骤

### 3. 交互式界面

#### 主页上传界面
- **磁性拖拽效果**：拖动文件时，上传区域会像磁铁一样吸引鼠标
- **双图并列**：源照片和目标照片并排展示
- **实时预览**：上传后立即显示预览
- **流畅动画**：所有操作都有丝滑的过渡效果

#### 结果展示页面
- **水平滑动导航**：顶部可滑动的卡片导航（模仿 AllTrails carousel）
- **卡片网格布局**：清晰的模块化展示
- **详细模态窗口**：点击卡片查看完整内容
- **阶段指示器**：显示当前处于第一阶段还是第二阶段

完全模仿 [DesignSpells.com](https://www.designspells.com) 的现代设计风格：

- **色彩系统**
  - 多彩渐变背景
  - 毛玻璃效果（backdrop-blur）
  - 柔和的阴影和边框

- **动画效果**
  - 页面切换的淡入淡出
  - 卡片的hover提升效果
  - 加载时的旋转动画
  - 按钮的闪光效果

- **布局风格**
  - 大量留白
  - 圆角设计（rounded-3xl）
  - 卡片式组件
  - 响应式网格

## 🎨 设计亮点

### 1. 磁性上传交互
当用户拖拽图片文件时：
- 上传区域会"感知"鼠标距离
- 在一定范围内产生磁性吸引效果
- 卡片会缩放、移动，仿佛被拉向鼠标
- 边框圆角也会动态变化

### 2. 水平滑动导航栏
在第二阶段显示的顶部导航：
- 卡片可以左右滑动
- 点击卡片会滚动到对应位置
- 活动卡片有底部指示线
- 左右箭头按钮智能显示/隐藏

### 3. 加载过渡动画
阶段切换时的全屏加载效果：
- 多层波浪动画
- 渐变色背景
- 旋转的进度指示
- 状态文字提示

### 4. 卡片交互
每个分析模块的卡片：
- Hover时轻微提升和缩放
- 背景渐变效果
- 图标旋转动画
- "NEW"标签（新解锁的卡片）

## 🔧 技术实现

### 核心技术栈
- **React 18** - 使用 Hooks 和函数式组件
- **TypeScript** - 完整的类型安全
- **Motion (Framer Motion)** - 强大的动画库
- **Tailwind CSS 4.0** - 现代CSS框架
- **Shadcn/ui** - 高质量UI组件

### 关键技术点

1. **Motion 动画**
   ```tsx
   <motion.div
     initial={{ opacity: 0, y: 20 }}
     animate={{ opacity: 1, y: 0 }}
     transition={{ delay: 0.1 }}
   >
   ```

2. **磁性效果实现**
   ```tsx
   const magneticX = useMotionValue(0);
   const magneticY = useMotionValue(0);
   // 根据鼠标距离计算吸引力
   ```

3. **两阶段状态管理**
   ```tsx
   const [showStage, setShowStage] = useState<'stage1' | 'loading' | 'stage2'>('stage1');
   ```

4. **横向滚动容器**
   ```tsx
   <div className="overflow-x-auto scrollbar-hide">
   ```

### 性能优化
- 使用 `AnimatePresence` 管理退出动画
- 懒加载大型组件
- 防抖处理滚动事件
- 合理使用 `useCallback` 和 `useMemo`

## 💡 核心价值

这个应用的核心价值在于：

1. **专业性**：基于真实的摄影理论和后期处理知识
2. **详细性**：每个参数都有具体的数值和说明
3. **可操作性**：生成的参数可以直接应用到软件中
4. **教育性**：帮助用户理解照片背后的技术原理
5. **高效性**：大幅缩短风格学习和复刻的时间
