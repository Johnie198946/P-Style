# 👤 用户头像 - 快速上手

## 🎯 一分钟了解头像功能

用户头像位于应用右上角，支持自定义上传、更换和移除。

---

## 📖 操作步骤

### 方法一：首次设置头像

```
第1步: 点击右上角的圆形头像按钮
        ↓
第2步: 在弹出菜单中点击「更换头像」
        ↓
第3步: 在对话框中点击「选择图片」
        ↓
第4步: 从电脑中选择一张照片
        ↓
第5步: 预览确认后点击「保存」
        ↓
完成！ 头像已更新
```

### 方法二：更换现有头像

```
第1步: 点击右上角的头像
        ↓
第2步: 点击「更换头像」
        ↓
第3步: 点击「选择图片」或点击头像预览区域
        ↓
第4步: 选择新照片
        ↓
第5步: 点击「保存」
        ↓
完成！ 头像已更换
```

### 方法三：移除头像

```
第1步: 点击右上角的头像
        ↓
第2步: 点击红色的「移除头像」选项
        ↓
完成！ 恢复为默认头像
```

## ✨ 特性说明

### 支持的图片格式
- ✅ JPG / JPEG
- ✅ PNG
- ✅ WEBP
- ✅ GIF (静态)

### 建议规格
- **尺寸**：400 x 400 像素
- **大小**：小于 2MB
- **形状**：自动裁剪为圆形

### 存储方式
- 💾 保存在浏览器本地（LocalStorage）
- 🔄 刷新页面后依然保留
- 🚫 清除浏览器数据会删除头像

## 💡 使用技巧

### 技巧 1：选择合适的照片
- 使用正方形或接近正方形的照片效果最佳
- 人物照片建议特写或半身照
- 确保主体在照片中央位置

### 技巧 2：快速更换
- 点击头像预览区域的相机图标可快速打开文件选择器
- 预览满意后直接点击保存，无需重新选择

### 技巧 3：恢复默认
- 如果不喜欢当前头像，使用"移除头像"功能
- 系统会恢复为默认的渐变色头像

### 技巧 4：保持一致性
- 在不同设备/浏览器上需要重新设置
- 建议保存照片文件以便快速设置

## ❓ 常见问题

### Q1: 为什么刷新后头像消失了？
**A**: 可能是浏览器的隐私模式或禁用了 LocalStorage。请检查浏览器设置。

### Q2: 可以上传多大的图片？
**A**: 建议不超过 2MB。过大的图片可能导致存储失败或性能问题。

### Q3: 头像为什么是圆形的？
**A**: 为了统一视觉风格，所有头像自动裁剪为圆形显示。

### Q4: 可以使用动图吗？
**A**: 当前仅支持静态图片。GIF 会显示第一帧。

### Q5: 如何在其他设备上使用同一头像？
**A**: 当前版本头像仅存储在本地。未来版本会支持云端同步。

### Q6: 更换头像后为什么看不到变化？
**A**: 尝试刷新页面。如果仍无变化，检查图片格式是否支持。

## 🎯 最佳实践

### 推荐头像类型
1. ✅ **个人照片**：专业形象照、证件照
2. ✅ **卡通形象**：个人喜欢的卡通角色
3. ✅ **风景照片**：标志性建筑、自然风光
4. ✅ **Logo图标**：个人品牌、团队标识
5. ✅ **艺术作品**：插画、设计作品

### 不推荐的类型
1. ❌ 分辨率过低的图片（模糊）
2. ❌ 长宽比例悬殊的图片（严重变形）
3. ❌ 包含敏感信息的图片
4. ❌ 版权受保护的图片

## 🔧 故障排除

### 问题：无法上传图片
**解决方案**：
1. 检查图片格式是否支持
2. 确认图片大小不超过限制
3. 尝试使用其他图片
4. 清除浏览器缓存后重试

### 问题：头像显示不完整
**解决方案**：
1. 使用正方形图片
2. 确保主体在图片中央
3. 尝试调整图片尺寸

### 问题：头像保存失败
**解决方案**：
1. 检查 LocalStorage 是否已满
2. 清除浏览器中的其他网站数据
3. 压缩图片大小后重试
