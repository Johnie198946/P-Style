# 内容管理系统使用指南

## 概述

QuantaNova 现已集成完整的内容管理系统，管理员可以在管理后台编辑和发布内容，这些内容会实时同步到主站显示。

## 功能特性

### 1. 主站订阅入口
- ✅ 在主站顶部导航添加了"订阅"按钮（位于"Log in"左边）
- ✅ 未登录用户点击可直接查看订阅计划
- ✅ 已登录用户点击进入个人订阅管理页面

### 2. 内容类型

管理后台支持发布以下类型的内容：

#### 文本内容 (text)
- 纯文本内容展示
- 支持标题、描述和正文
- 可添加外部链接

#### 图片内容 (image)
- 图片展示
- 支持图片标题和描述
- 可添加点击跳转链接

#### 横幅广告 (banner)
- 页面顶部横幅展示
- 支持大标题、描述和图片
- 可关闭功能
- 渐变背景装饰

#### 功能介绍 (feature)
- 卡片式展示
- 适合展示产品功能特性
- 支持图文混排

#### 公告通知 (announcement)
- 醒目的公告样式
- 带有动态指示点
- 用户可关闭

### 3. 显示位置

内容可以设置显示在以下位置：
- **首页 (home)**: 主站首页
- **结果页 (results)**: 分析结果展示页
- **订阅页 (subscription)**: 订阅计划页面
- **用户中心 (user-center)**: 用户个人中心
- **全站 (all)**: 所有页面显示

### 4. 管理后台操作

#### 访问管理后台
使用快捷键 `Ctrl + Shift + A` 打开管理员登录对话框

#### 内容管理功能
1. **新建内容**
   - 点击"新建内容"按钮
   - 选择内容类型
   - 填写标题、描述等信息
   - 上传图片（如需要）
   - 勾选"立即发布"即可在主站显示

2. **编辑内容**
   - 在内容列表点击"编辑"按钮
   - 修改内容信息
   - 保存后立即在主站更新

3. **发布/取消发布**
   - 点击内容卡片上的"发布"/"取消发布"按钮
   - 已发布的内容会显示在主站
   - 草稿状态的内容不会显示

4. **删除内容**
   - 点击内容卡片上的删除按钮
   - 确认后永久删除

5. **批量操作**
   - 选中多个内容
   - 批量删除

6. **搜索和筛选**
   - 按内容类型筛选
   - 搜索标题或描述关键词

### 5. 数据存储

- 使用 localStorage 存储内容数据
- 管理后台和主站实时同步
- 支持订阅更新通知

## 技术实现

### 核心组件

1. **ContentEditor** (`/components/admin/ContentEditor.tsx`)
   - 内容编辑器组件
   - 支持文本、图片上传
   - 表单验证

2. **ContentStore** (`/lib/contentStore.ts`)
   - 内容数据存储服务
   - 单例模式
   - 支持订阅/发布模式

3. **PublishedContent** (`/components/PublishedContent.tsx`)
   - 主站内容展示组件
   - 根据位置和类型自动加载
   - 响应式布局

4. **ContentManagement** (`/components/admin/ContentManagement.tsx`)
   - 管理后台内容管理页面
   - 内容列表展示
   - CRUD 操作

### 数据流

```
管理后台 (ContentManagement)
    ↓
内容编辑器 (ContentEditor)
    ↓
内容存储 (contentStore)
    ↓
主站组件 (PublishedContent)
    ↓
用户看到的内容
```

## 使用示例

### 在主站添加内容展示

```tsx
import { PublishedContent } from './components/PublishedContent';

// 显示首页的公告
<PublishedContent position="home" type="announcement" />

// 显示订阅页的功能介绍
<PublishedContent position="subscription" type="feature" />

// 显示所有已发布的横幅
<PublishedContent position="home" type="banner" />
```

### 创建新内容（管理后台）

1. 进入管理后台 → 内容管理
2. 点击"新建内容"
3. 选择内容类型（如"功能介绍"）
4. 填写信息：
   - 标题：AI 智能分析
   - 描述：深度学习算法，精准识别照片风格
   - 正文：详细描述功能特点...
   - 显示位置：首页
   - 勾选"立即发布"
5. 点击"保存并发布"

内容会立即显示在主站首页！

## 最佳实践

### 内容策略

1. **首页内容**
   - 使用横幅展示重要促销或活动
   - 使用功能介绍卡片展示产品特性
   - 及时发布公告通知

2. **订阅页内容**
   - 突出订阅优势
   - 展示成功案例
   - 提供优惠信息

3. **图片规范**
   - 横幅图片：建议 1200x400 像素
   - 功能图片：建议 800x600 像素
   - 文件大小：不超过 5MB
   - 格式：JPG、PNG、GIF

4. **文案建议**
   - 标题：简短有力，10-20字
   - 描述：一句话概括，30-50字
   - 正文：详细说明，100-300字

## 注意事项

1. **数据持久化**
   - 当前使用 localStorage，浏览器清除缓存会丢失数据
   - 生产环境建议连接真实后端 API

2. **权限控制**
   - 管理后台需要管理员权限
   - 建议加强身份验证和授权机制

3. **性能优化**
   - 大量内容时考虑分页加载
   - 图片建议使用 CDN 加速

4. **SEO 优化**
   - 动态内容可能影响 SEO
   - 考虑服务端渲染方案
